# Nabla uses Jekyll
Build a Jekyll blog in minutes, without touching the command line.

## Post formatting

Example of a post:

```console
---
layout: post
title: "Here goes the title"
tags: [tag1, second tag, third, etc]
comments: true
link: "http://example.com/news/126"
canonical: "example.com"
excerpt: "<p>Bla bla bla...</p>"
---
```
